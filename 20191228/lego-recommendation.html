聪哥聊玩具之乐高推荐

玩具 乐高 积木 2019/12/28

乐高这家于 1932 年开始创办的公司，在长达大 86 年风吹雨打过程中，以最原始的积木玩具的形式，经历了书籍、电影、电子游戏、互联网、VR、人工智能等娱乐革命的浪潮，跟他们比，乐高就像古董一样原始，为了保持盈利，半个世纪以来，整个公司的业务在不断地创新：

而它某些经典款的收益率除开房子和比特币这些特殊品种以外，可能就无出其右了，价格翻十倍是经常出现的事情；下面我就推荐一些值得购买和收藏的乐高积木和系列，价格仅供参考。

单个 set

10179 千年隼

价格：10000 左右
所属系列：星球大战
颗粒数：5197
推荐理由：经典 IP，但是在 75192 发布之后，就没有了竞争力了
推荐指数：★★★★★

价格：5000 元左右
所属系列：星球大战
颗粒数：7541
推荐理由：重要 IP，相比 10179，颗粒数更多，尺寸更大，细节更丰富，可活动的机关，人仔配置豪华；最佳乐高，毫无悬念地落在了千年隼的身上，作为乐高商品史一系列记录的创造者和保持者，或许在接下来的10年、20年都没有产品可以超过了
推荐指数：★★★★★

系列

乐高每年会有新品的发布，老产品的生产线会被关闭，这样就产生了一些绝版乐高。下面一些很容易绝版的乐高系列

街景系列

排在第一位的自然是街景系列，这是一个从 2009 年的 10197 消防局开始的系列，到 2019 年已经有 10 年的时间了：

2007年——2009年的三款就被街景粉称为”老三绝”。“老三绝”分别是街角咖啡馆（10182）、市场街（10190）和绿色杂货店（10185），用现在的眼光看, 这三款街景拼砌方式还很简单，内饰缺乏，可玩性一般。但是当年并没有这么多可以新品所以就显得特别珍贵。“老三绝”的价格一路飞涨，三款当年官方售价不过千元的乐高，在现在某宝价格近2w，涨幅约620%。

到后来09-12年发布的三款：消防局（10197）、百货大楼（10211）、市政厅（10224）也绝版了，被称为“新三绝”。

  • 2009的乐高消防局里面最经典的设计是升降电梯，用积木做的电梯每层都可以停靠。
  • 2010年的百货商店是一套极具法国风情的三层百货公司。
  • 2012年市政厅是目前街景系列中最高的SET，楼内的电梯可移动。

新三绝的价格没有老三绝那么夸张，但也和原版比起来也上涨不少，三套一起大概要一万多。

欢迎来到有趣的微观世界。街景系列很多细节真的藏着“魔鬼”般神奇，不禁感叹开发者的奇思妙想，具体可以查看：有哪些值得拥有的乐高系列? - 谢谢侬的回答 - 知乎

这是一个将每个城市好几个有特色的标志性建筑打包集中在一个 set 里的系列，每个 set 都只有几百块积木，虽然积木数量不多，但是尽可能的展现了建筑的主要特征，非常巧妙和难得；另外，这个系列还有一个特别棒的一点就是：非常适合用来展示，比如以下图片来自 Flickr 网友 liqiwei2000：

创意百变系列

这个系列的积木入手价格多数在 200 以内，遇到好价的情况下，甚至能低至七八十；同时玩法丰富，除了官方自带 3 个造型外，多数都被开发出其他 MOC 造型，可玩性很高，考验玩家的想象力和创造力，具体可以查看：双11什么乐高值得买：这5款乐高创意百变，百元价位，绝对值得入

小技巧

  1. 不是大佬，不是喜欢到必买不可，不要第一手买，后面都会有好价的。
  2. 尽量在电商大促的时候买自己喜欢的套装或价格的确有优势的套装。
  3. 不要为了赠品高价买自己不喜欢的乐高。真的喜欢赠品可以在tb上直接买啊
  4. 星战系列，超级英雄系列大多数买人仔就可以了
  5. 对于有比较吸引人的人仔套装要多留意，这种套装一般升值潜力较大
  6. ideas 系列喜欢就买吧，每一个都很有创意
