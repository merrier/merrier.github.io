4.任何一个函数都有一个prototype属性，指向其原型，比如一个函数fn，那么fn.prototype是一个对象，同时满足fn.prototype.constructor === fn 5.所有通过函数new出来的东西，这个东西都有一个___proto__指向这个函数的prototype：

var arr = []; //等价于 var arr = new Array()
arr.__proto__ === Array.prototype; //true

当你想用一个对象（或者一个数组）的某个功能时：如果该对象本身具有这个功能，则直接使用；如果该对象本身没有这个功能，则去__proto__中找
