## imgcook

官网：https://www.imgcook.com/

由设计稿一键只能生成代码，有 Sketch 版插件和 Photoshop 版插件，同时还可以在线生成

## Github Mobile

官网：https://github.com/mobile

支持 iOS 和 安卓，万众期待的 Github 移动版终于发布了

## TexturePacker

官网：https://www.codeandweb.com/texturepacker

主要用于游戏开发时的不同元素的组织，同时可以在不减少文件本身体积的前提下进行优化

## 写作助手

Github 地址：https://github.com/ystcode/BlogHelper

帮助国内用户写作的托盘助手，一键发布本地文章到主流博客平台（知乎、简书、博客园、CSDN、SegmentFault、掘金、开源中国），剪贴板图片一键上传至图床（新浪、Github、图壳、腾讯云、阿里云、又拍云、七牛云）

## 墨刀

官网：https://modao.cc/

支持 macOS、Windows、Ubuntu、iOS 和 Android，同时还有 Sketch 插件；在家远程办公的好帮手，更是一款专业、可靠、简洁、易懂的在线产品原型工具与产品设计团队协作平台，能高效、轻松的解决团队成员远程办公，高效协作编辑，实时项目协作管理，信息同步，沟通透明且高效，企业成员项目权限灵活管理，可直接复用企业共享素材库，简，金融级保障数据安全。

提供免费下载软件的网站
===========

*   异次元：https://www.iplaysoft.com/，该站是一个老站，历史悠久，站长是一名网页设计师！网站主要专注于推荐一些实用软件、App 应用和互联网优质资源，并且其推荐都比较良心，每篇都有详细的图文评测，而且还提供软件资源下载。另外，该站有时也会提供一些正版软件的优惠码和打折信息，你们也可以关注下。
*   Awesome Mac：http://wangchujiang.com/awesome-mac/index.zh.html，该站主要专注于推荐 Mac 电脑上一些好用的软件，涵盖的范围非常之广，如果你是苹果产品重度使用者，这个站点你必须要收藏！
*   zdfans：http://www.zdfans.com/，该站汇集的优质软件非常多，而且领域范围也很广。绝大多数的绿色软件都可以在这里面找到，虽然此前这个网站经历了卖站风波，但目前来看，其更新速度和频率依然都很不错！很多软件都可以在这里找到，例如 IDM，亿图图示，PotPlayer 等，五星级推荐大家收藏！
*   酷安：https://www.coolapk.com/，酷安是一个涵盖了各类小众和超实用 App 的聚宝盆。除了推荐一些常用的 App 和游戏之外，酷安最主要的是会给大家推荐一些小众 App，而且其手机端 App 的发现频道也有很多好玩实用的 App 推荐，平时没事多逛逛，相信你能淘到不少宝贝。
*   最美应用：http://zuimeia.com/，该网站主要专注于为大家推荐和分享安卓和 IOS 端的 App，网站每天会推荐一个各领域的好用 App 或者游戏。最贴心的是里面还有众多高手达人分享着他们的使用经验和心得，这为大家选择使用 App 提供了非常多的建议。
*   少数派：https://sspai.com/，少数派的口号是：运用数字产品或科学方法,帮助用户提升工作效率和生活品质。该站推荐的网站和工具相对来说比较精品，每一种工具都有较为详细的测评。除此之外，该站比较吸引我的地方就是它比较注重方法和理论的讲解，而不是仅仅局限于表面的技巧。另外，网站也有一些关于电子产品的使用技巧，而且还有很多隐藏的专业高手达人！下次你想寻找某方面的信息时，不妨先去这个网站看看，例如搜索【小程序】，你会发现有 43 篇讲解小程序的文章
