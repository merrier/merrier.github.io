## 网页截屏

强大的 Chrome 内置了截屏的功能：

1. 首先打开 Chrome 控制台（打开快捷方式 `Option` + `Command` + `i`）
2. 打开命令行（打开快捷方式 `shift` + `cmd` + `p` 或 win 系统 `shift` + `ctrl` + `p`）
3. 其次在命令行输入 `capture` 就可以看到

> 注意：capture full size screenshot 和 capture screenshot 的区别在于前者是整个页面完整信息的截图，后者只是当前页面的截图
