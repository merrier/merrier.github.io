## 中文排版

可以参考 Github 上面的[中文文案排版指北](https://github.com/sparanoid/chinese-copywriting-guidelines)，主要有以下几点规范：

* 中英文之间需要增加空格
* 中文与数字之间需要增加空格
* 数字与单位之间需要增加空格
* 全角标点与其他字符之间不加空格
* 不重复使用标点符号
* 使用全角中文标点
* 数字使用半角字符
* 遇到完整的英文整句、特殊名词，其内容使用半角标点
* 专有名词使用正确的大小写
* 不要使用不地道的缩写
* 链接之间增加空格

## 图片排版

建议图片都居中展示，同时在 markdown 中建议采用下面的方式插入图片：

```html
<div align='center'><img src='' alt='' width='400'/></div>
```

最好指定宽度，这样可以避免某些图片原图尺寸过大时比较难看的问题。

## 避免纯文字

同样的内容，视频表达的信息要比图文多得多；但是在个人博客中，出于对读者流量的考虑，图文并茂是更好地展现形式，相比纯文字会更加生动形象。

所以，在写个人博客时，需要注意避免纯文字，通过图片的辅助使文章变生动立体。
