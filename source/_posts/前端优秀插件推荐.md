这里集中了一些我在项目中或平常看到听到的好的插件，话不多说，直接干货：

| 插件名称 | 描述 | 地址 | 备注  | 依赖   |
|--------|------|-----|------|--------|
| turn.js | 能够将HTML内容组织成看起来像一本真正的书籍或杂志，用到HTML5的所有优点 | [Github](https://github.com/blasten/turn.js) &#124; [官网](http://www.turnjs.com/) | 可以工作在旧浏览器中比如：IE 8 |  |
| Tdrag | 拖拽的插件，可以拖拽任意元素 | [Github](https://github.com/tezml/Tdrag/) &#124; [Demo](http://www.jq22.com/yanshi8362)  |  | jQuery |
| tagsinput | 动态标签生成，并添加了删除按钮和其他好看的样式 | [Github](https://github.com/xoxco/jQuery-Tags-Input) &#124; [官网](http://xoxco.com/projects/code/tagsinput/) &#124; [Demo](http://xoxco.com/projects/code/tagsinput/example.html)  |  | jQuery |
| slimScroll | 支持把内容放在一个盒子里面，固定一个高度，超出的则使用滚动。 | [Github](https://github.com/rochal/jQuery-slimScroll) &#124; [Demo](http://www.jq22.com/yanshi1425)  | 不过不支持resize的时候重新调用插件，有一个贡献者已经修改过源码，不过作者并没有加进去。插件项目地址：[插件地址](http://rocha.la/jQuery-slimScroll)（插件不支持IE7） | jQuery |
| scratchit.js | 一款基于HTML5 canvas的jQuery刮刮卡效果插件。这个刮刮卡效果使用两张图片和canvas来制作，可以使用鼠标擦除上面的图片，将下面的图片显露出来，模拟现实的刮刮卡效果 | [Demo](http://www.htmleaf.com/Demo/201506071985.html)  | 可以用来做手机游戏刮刮卡，亲测可用 | jQuery |
| myFocus | 可以实现图片轮播，风格多种多样 | [官网](http://demo.jb51.net/js/myfocus/) &#124; [Demo](http://demo.jb51.net/js/myfocus/)  | 没有 Github 地址，因为人家是用 SVN 的 |  |
| move.js | 模拟CSS3动画的js插件。该插件可以完成CSS3的各种动画效果：移动、变形、倾斜、背景色渐变、旋转等。并且可以在动画中使用各种easing效果 | [Github](https://github.com/visionmedia/move.js/) &#124; [Demo](http://www.htmleaf.com/Demo/201501281292.html)  |  | jQuery |
| mCustomScrollbar | 滚动条样式插件，可以自由的定制自己喜欢的滚动条 | [Github](https://github.com/malihu/malihu-custom-scrollbar-plugin) &#124; [官网](http://manos.malihu.gr/jquery-custom-content-scroller/) &#124; [Demo](http://manos.malihu.gr/repository/custom-scrollbar/demo/examples/complete_examples.html)  | 兼容IE6和IE7等的浏览器 |  |
| Lightbox | 实现图片浏览效果，具有的功能包括：自动根据窗口的大小缩放图片，模式窗口，幻灯片方式播放，内容预加载，渐变等效果 | [Github](https://github.com/lokesh/lightbox2/) &#124; [官网](http://lokeshdhakar.com/projects/lightbox2/) &#124; [Demo](http://www.zhangxinxu.com/jq/balupton_zh/demo/)  | 有基于 jQuery 的，有基于 Prototype 的，也有使用原生 JavaScript 写的 |  |
| html2canvas | 将html的内容以canvas形式生成图片，亲测好用 | [Github](https://github.com/niklasvh/html2canvas) &#124; [官网](http://html2canvas.hertzen.com/) &#124; [Demo](http://html2canvas.hertzen.com/examples.html)  |  |  |
| gridly | 一个 jQuery 插件，它能够拖动，删除和调整一个网格的大小，亲测好用 | [Github](https://github.com/ksylvest/jquery-gridly) &#124; [官网](http://ksylvest.github.io/jquery-gridly/) &#124; [Demo](http://www.jq22.com/yanshi386)  |  | jQuery |
| FullPage | 可以做出全屏网站 | [Github](https://github.com/powy1993/fullpage) &#124; [Demo](http://www.dowebok.com/demo/2014/77/)  | 兼容桌面端(ie5.5+) 和手机端 |  |
| fine-uploader | 采用 ajax 方式实现文件上传，同时在浏览器中直接支持文件拖拽 | [Github](https://github.com/valums/file-uploader) &#124; [官网](https://fineuploader.com/) &#124; [Demo](https://fineuploader.com/demos)  | 对浏览器版本有要求类似 IE 版本必须是 9 或是更高的 IE10 |  |
| csshake-master | CSS3 元素抖动动画库 | [Github](https://github.com/elrumordelaluz/csshake) &#124; [官网](http://elrumordelaluz.github.io/csshake/) &#124; [Demo](http://www.ui3g.com/demos/show/1233/)  |  |  |
| ajaxfileupload | 一个异步上传文件的 jQuery 插件 | [Github](https://github.com/davgothic/AjaxFileUpload) |  | jQuery |
| animate.css | 齐全的 CSS3 动画库 | [Github](https://github.com/daneden/animate.css/) &#124; [Demo](http://www.jq22.com/yanshi819)  |  |  |
| css3d-engine | CSS3d 动画，主要是在 transform 上做文章 | [Github](https://github.com/shrekshrek/css3d-engine) &#124; [官网](http://css3d.bitworking.de/) &#124; [Demo](http://css3d.bitworking.de/examples/example01.html)  | 可以用来做720度或320度全景，亲测可用 |  |
| hammer.js | 可以完美的实现在移端开发用到的大多数事件，如：点击、滑动、拖动、多点触控等事件 | [Github](https://github.com/hammerjs/hammer.js/) &#124; [官网](http://hammerjs.github.io/) &#124; [Demo](http://codepen.io/runspired/full/ZQBGWd/)  |  |  |
| iscroll.js | 移动端广泛应用的插件之一，用于让元素在固定高度盒子内滚动 | [Github](https://github.com/cubiq/iscroll) &#124; [官网](http://cubiq.org/iscroll-4) &#124; [Demo](http://lab.cubiq.org/iscroll/examples/simple/)  | 版本 4 还包括缩放、拉动刷新、精确捕捉元素、自定义滚动条等功能 | jQuery |
| touchSwipe.js | 和 hammer.js 类似，同样是移动端触屏触摸特效插件 | [Github](https://github.com/mattbryson/TouchSwipe-Jquery-Plugin) &#124; [官网](http://labs.rampinteractive.co.uk/touchSwipe/demos/) &#124; [Demo](http://labs.rampinteractive.co.uk/touchSwipe/demos/Enable_and_destroy.html)  | 有一个 bug：手指事件会在 a 标签超链接内失效 | jQuery |
| jquery.validate | 表单验证插件，非常方便实用 | [Github](https://github.com/jquery-validation/jquery-validation) &#124; [官网](https://jqueryvalidation.org/) &#124; [Demo](https://jqueryvalidation.org/files/demo/)  |  | jQuery |
| jstween | 将某一个元素粒子化 | [Github](https://github.com/shrekshrek/jstween) &#124; [Demo](http://shrek.imdevsh.com/demo/performance/test1/jstween.html)  | 具体应用还未尝试，但是该插件的作者有许多优秀的项目，可以 follow 一下 |  |
| DlHighlight | 代码高亮脚本，只支持 4 种编程语言：JS、CSS、XML、HTML | [Github]() &#124; [官网](http://mihai.bazon.net/projects/javascript-syntax-highlighting-engine) &#124; [Demo](http://demo.jb51.net/js/2012/hl/)  |  | jQuery |
